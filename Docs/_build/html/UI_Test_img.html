UI_Test_img

class UI_Test_img.ImageSelection

Bases: PyQt5.QtWidgets.QWidget

User Interface for blind comparison of images. The same image plane is shown deconvolved with two different algorithms. The user is asked to select the plane he/she likes better. In the end, statistics are returned that reflect on how often each deconvolution method was selected over the other ones.
